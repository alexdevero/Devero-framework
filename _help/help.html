Fonts

If you want to host the fonts on your own server instead of Google's CDN (Content Delivery Network), download and the font Raleway and Righteous font from the CDN.

Otherwise, the fonts will be loaded from Google Font CDN as default to lower number of connection to your web server and make your web load faster.

Template

CUBE is fully responsive HTML5 template. HTML and CSS code is well comented and easy to understand. Group elements are highlighted at the beginning and also at the end

Basic HTML code structure:

CSS and Sass files for styling the Template can be found in "styles" folder.

Basic CSS code structure:

All JavaScript files, including jQuery plugins can be found in "js" folder. Inside this folder is also folder "vendor" containg offline version of jQuery 2.1.4.

For your own JavaScript code, in "js" folder is prepared file "scripts.js".

Images are included in "images" folder and ready to use.

Every image is optimized to lower the bandwith. All images are styled through CSS, so you don't need to worry when replacing default images with your own.

Template width is 960px and it is fully responsive.
